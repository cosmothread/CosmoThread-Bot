# CosmoThread-Bot
Zero-cost X bot for Grok-style multiverse thread sims. Dino portals await
AI-powered X thread weaver: Turn prompts into branching multiverse sims with dino lore & physics odds.

## Quickstart
1. Fork & clone.
2. Add X API keys to .env.
3. `pip install -r requirements.txt` (in Replit).
4. Run `python bot.py`.

Co-built with Grok. Universe-grokking since Nov 2025.
